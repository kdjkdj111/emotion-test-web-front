# 🎭 Emoticon Lab - Front-end
> 카카오 이모티콘 규격 검증 및 실시간 듀얼 시뮬레이터 UI

본 프로젝트는 제작한 이모티콘이 실제 카카오톡 환경에서 어떻게 보일지 실시간으로 시뮬레이션하고, 제안 규격에 맞는지 백엔드와 통신하여 검증하는 도구입니다.

## ✨ 주요 기능
- **Landing Page**: 서비스 소개 및 진입점
- **Validation Grid**: 최대 32개의 이모티콘을 한눈에 업로드하고 상태 확인
- **Detail Analysis**: 개별 이미지의 해상도, 용량, 타입 및 검증 결과 상세 조회
- **Dual Simulator**: 실제 모바일 채팅창 UI를 통해 송/수신 환경 실시간 시뮬레이션

## 🛠 Tech Stack
- **Library**: React.js
- **Styling**: Tailwind CSS
- **API Client**: Axios
- **State Management**: React Hooks (useState, useEffect, useRef)
